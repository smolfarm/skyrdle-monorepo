# Skyrdle

Skyrdle is an open source word game you can play on your AT Protocol account, storing your scores in your PDS! The [official implementation is maintained at skyrdle.com](https://skyrdle.com). Relies on Mongo in order to store a list of words as well as canonical game scores.

# Sub-projects

* `skyrdle-admin` - Admin tools to help manage the game
* `skyrdle-native` - React Native app for iOS & ANdroid
